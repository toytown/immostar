What is Jiwhiz Blog?

A small blog application developed by Yuan Ji.

Everything should be made as simple as possible, but not one bit simpler.

Albert Einstein

This is a simple blog website I build with all open source projects, to experience the Java architecture design for Web and mobile applications. The whole system is built upon Spring Framework, using SpringMVC, Spring Security, Spring Social, Spring Data, and some other open source projects, like Thyemleaf, Mockito, etc. All data is stored in a NoSQL database - MongoDB, and front end UI is using Twitter Bootstrap with jQuery. The Web application is deployed to Cloud Foundry (Running at AppFog.com), an open source PAAS cloud platform.

The whole application code is put at GitHub: github.com/jiwhiz/JiwhizBlogWeb under Apache License Version 2.0.
